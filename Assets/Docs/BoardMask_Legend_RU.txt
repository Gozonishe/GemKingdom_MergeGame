BOARD MASK — ОБОЗНАЧЕНИЯ ДЛЯ ПОСТРОЕНИЯ ПОЛЯ

Направление чтения:
- строки записываются сверху вниз;
- символы внутри строки идут слева направо;
- количество строк должно совпадать с Board Rows;
- количество символов в каждой строке должно совпадать с Board Columns.

Обозначения:

1 или .  — обычная свободная клетка поля.
0 или _  — отверстие: клетка в этой позиции не создаётся.
S        — клетка с целым камнем (Full Stone).
C        — клетка с треснувшим камнем (Cracked Stone).

Для S и C регистр не важен, но для единообразия рекомендуется использовать
заглавные буквы.

Пример маски для поля 6 x 6:

001100
011110
11SC11
11CS11
011110
001100

Важно:
- внутри строки нельзя ставить пробелы или другие символы;
- пустые строки между рядами игнорируются, но лучше их не добавлять;
- если Board Mask оставить пустой, будет создано полное прямоугольное поле;
- если маска имеет неправильный размер или неизвестный символ, во время игры
  будет использовано полное прямоугольное поле, а Unity выведет предупреждение.
