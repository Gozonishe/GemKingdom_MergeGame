Production-like mobile merge game
The goal was to create a production-like mobile live event game:

*Main Menu → Event Button → Event Popup → Close / Return*

- Unity UI / uGUI implementation
- Mobile portrait layout
- Main menu HUD assembly
- Side event buttons
- Modal live event popup
- Background dim overlay
- Button interaction flow
- Lightweight VFX integration
- UI hierarchy organization
- Visual polish for a mobile F2P interface

Tools:

- Unity UI / uGUI

- Photoshop
